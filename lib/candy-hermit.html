CandyHermit

Fuzzy finder overlay for terminal UIs

port of Genekkion/theHermit fuzzyfinderoverlaysearch

Fuzzy-search overlay — type to filter a list, arrow keys to select, Enter to confirm. Ships as a model wrapper that composes into any bubbletea program.

Install

composer require sugarcraft/candy-hermit

Quickstart

use SugarCraft\Hermit\Hermit;

$hermit = Hermit::new(
    items: ['Ruby', 'PHP', 'Go', 'Rust', 'Python'],
    prompt: 'Choose language: ',
);

What's in the box

Fuzzy matchingFuse.js-style fuzzy search on item text.
Hermit modelFull bubbletea Model with init / update / view.
ConfigurableCustom formatter, header text, empty message.
