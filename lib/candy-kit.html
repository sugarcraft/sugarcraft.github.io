CandyKit

CLI presentation helpers

StatusLine (✓/✗/⚠/ℹ glyph + message), Banner (rounded title block), Section / Stage (numbered build-script lines), HelpText (fang-style --help page). Library-only, no Symfony Console required.

Install

composer require candycore/candy-kit
