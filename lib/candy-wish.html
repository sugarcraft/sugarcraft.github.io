CandyWish

SSH server middleware framework

Logger, Auth (user + key fingerprint allowlist), RateLimit (per-IP token bucket), BubbleTea (mount a CandyCore Program). Leans on host sshd via ForceCommand — battle-tested key exchange, fail2ban, audit log, all stay with OpenSSH.

Install

composer require candycore/candy-wish
