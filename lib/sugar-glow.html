SugarGlow

Markdown CLI viewer / pager

Renders to stdout (default) or opens a fullscreen Viewport pager (-p / --pager). All eight CandyShine themes available, custom JSON themes via --theme-config, --no-hyperlinks toggle.

Install

composer require candycore/sugar-glow
