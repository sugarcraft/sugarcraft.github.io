SugarSpark

ANSI escape-sequence inspector

Labels SGR (incl. truecolor), CSI cursor / erase / mode toggles, OSC (title / hyperlink / palette / progress), DCS (XTVERSION / sixel), APC (CandyZone, kitty graphics), SS3, 2-byte ESC. Unrecognised sequences fall back to a generic descriptor — nothing is silently swallowed.

Install

composer require candycore/sugar-spark
